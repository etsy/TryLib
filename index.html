TryLib PEAR Channel

Registering the channel: 
pear channel-discover etsy.github.com/TryLib
Listing available packages: 
pear remote-list -c TryLib
Installing a package: 
pear install TryLib/package_name
Installing a specific version/stability: 
pear install TryLib/package_name-1.0.0
pear install TryLib/package_name-beta
Receiving updates via a feed: 
http://etsy.github.com/TryLib/feed.xml

The etsy.github.com/TryLib PEAR Channel Server is proudly powered by Pirum 1.1.5
